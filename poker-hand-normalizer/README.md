## Poker Hand History Parser (Python)

Parser desarrollado en Python para el procesamiento de historiales de manos de póker.

### Funcionalidades principales
- Anonimización de nombres de jugadores para preservar la privacidad.
- Normalización y corrección de formatos de divisas.
- Eliminación de caracteres no estándar y problemas de codificación.
- Limpieza y estandarización de datos para facilitar su consumo posterior.

### Objetivo
Este proyecto se utiliza para generar conjuntos de datos limpios y consistentes, destinados a probar y validar el funcionamiento de un parser de póker online.

### Uso
Ideal para flujos de procesamiento de datos, pruebas automatizadas y validación de sistemas que requieren historiales de manos de póker normalizados.
